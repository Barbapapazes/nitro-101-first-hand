# Nitro 101 - First hand

[Read my article (FR)](https://esteban-soubiran.site/articles/nitro-101-premire-prise-en-main/) to learn Nitro !

## Install dependencies

```bash
pnpm install
```

## Run the app

```bash
pnpm run dev
```

## Build the app

```bash
pnpm run build
```

## Run the app in production

```bash
pnpm run preview
```
